Overview

Company Inspector is a Flask-based web application designed to streamline the management and verification of employees and teams within a company. Tailored for administrative use, the app offers powerful tools for company admins to efficiently organize and supervise their workforce. Additionally, it provides verification capabilities for external users, allowing them to authenticate employee and team details. With "Company Inspector," admins can easily manage employee records, track team compositions, and oversee day-to-day organizational structures. The verification feature ensures that both internal and external parties can quickly validate the authenticity of team members and employee credentials, promoting transparency and trust within the company and with external partners. The app combines intuitive management tools with robust security and verification features, making it an essential resource for companies looking to optimize their employee and team oversight processes.

Features

  • Employee and Team Management: Easily manage employee records, track team compositions, and supervise organizational structures.
  • External Verification: Allow external users to verify employee and team details, promoting transparency and trust.
  • Admin Tools: Powerful tools to organize and monitor your workforce with ease.
  • Security Focused: Robust security features to ensure data integrity and secure verification.
  • Export Employees Table to Excel: Download employee data in Excel format for easier offline access and analysis.
  • Export Employees Table to PDF: Generate PDF reports of your employee data for formal documentation or sharing.
  • Import Employees as Excel: Quickly upload bulk employee data via Excel files, streamlining data entry and updates.
  • Role-Based Access: Supports 1 Super Admin and up to 6 Mini Admins, with the Super Admin responsible for approving and controlling these Mini Admins.

Getting Started

This section will help you get started with setting up and running the project on your local machine.

Setting Up Locally

  1. After Downloading the software

    Locate the zip file from the asserts you downloaded company_inspector.zip and copy it to your preferred location. Once you have placed it in a convenient location where you can easily access it using the terminal, follow these steps depending on your operating system:

  2. Open The Terminal or Command Prompt

    Windows
    1. Click on the Start Menu.
    2. Type cmd in the search bar and hit Enter.
    macOS
    1. Open Finder.
    2. Go to Applications > Utilities.
    3. Double-click Terminal.
    Linux
    1. Press Ctrl + Alt + T on your keyboard.
    2. Alternatively, search for Terminal in your applications menu.

  3. Change Dir To Zip

    you will need to change directory to where you placed the zip file

    Windows 
    cd C:\path\to\zip\location
    macOS & Linux 
    cd /path/to/zip/location

  4. Unzip the zip file

    Windows

    Use the following command to unzip a file:

    tar -xf company_inspector.zip

    Alternatively, use PowerShell:

    Expand-Archive -Path company_inspector.zip
    macOS & Linux 
    unzip company_inspector.zip

  5. Change Dir To Project Directory

    Windows

    After unzipping, change to the unzipped folder:

    cd C:\company_inspector
    macOS & Linux 
    cd /company_inspector

  6. Created a Virtual Environment

    Windows, macOS & Linux 
    python -m venv venv

    OR 
    python3 -m venv venv

  7. Activating the Virtual Environment

    Windows 
    venv\Scripts\activate
    macOS & Linux 
    source venv/bin/activate
    8. Then from there we move on to the next Step

  8. Install Requirements

    Install the necessary Python packages:

    pip install -r requirements.txt

  9. Set Up the Database

    Initialize the database and apply migrations using Flask-Migrate: 

    flask db upgrade

  10. Run the Application

    Start the Flask development server:

    python main.py

Administrator Guide

After the server starts successfully, the main process will begin. As mentioned earlier, authentication is part of this app, so you will first need to sign up as a super admin to be authenticated.

Authentication

The super admin is the primary driver of this app, followed by co-admins. The super admin is responsible for approving co-admins, and only 10 admins, including the super admin, are allowed.

Creating a Super Admin

Navigate to /auth/admin/super/signup/
Then, fill in the provided form.

Signing Back into the App

Navigate to /auth/admin/signin/
Then, fill in the provided form.

Requesting Admin Permission

Navigate to /auth/admin/signup/
Fill in the provided form. It doesn’t matter whether you are an existing employee or not, as long as there are fewer than 10 admins.
Then wait for the super admin to approve your request. The sign-in process is shown above.
Once you have access to the dashboard which is found at /admin/, you can add employees and group them.

User Guide

To start verifying, no authentication or authorization is required. Simply navigate to /, where you will find two tabs: Verify Employee, which can be used to verify a single employee, and Verify Team, which can be used to verify a team as a whole.
In either the Employee or Team verification input, just enter an employee's or team's Alpha code respectively, and then hit "Verify" to get the results.
